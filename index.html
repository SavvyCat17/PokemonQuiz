The goal of this quiz-style game is to determine which move type out of four random types will deal the most to the Pokemon you've encountered.

If you make the wrong choice three times, the game is over.

The type matchups used for this game are based on the latest update to the Pokemon type chart, but they do not account for Pokemon abilities that affect type matchups.

To get started, select a Pokemon generation and hit Play. Good luck!
